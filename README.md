# 🍽️ Gemma Restaurant Bot

An AI-powered restaurant chatbot built using **FastAPI**, **Gemma 3**, and **Ollama**.  
This project allows users to interact with a restaurant assistant that can answer menu-related questions, provide recommendations, and simulate customer support conversations in real time.

---

## 🚀 Features

- 🤖 AI-powered chatbot using Gemma 3
- ⚡ FastAPI backend for API handling
- 💬 Interactive chat interface
- 🧠 Natural language understanding
- 🍕 Restaurant menu assistance
- 📦 Ollama integration for local LLM execution
- 🔄 Real-time chatbot responses

---

## 🛠️ Technologies Used

- Python
- FastAPI
- Ollama
- Gemma 3
- Uvicorn
